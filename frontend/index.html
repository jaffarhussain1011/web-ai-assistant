Live Support with AI

Instant answers for your customers — powered by AI, running on your server

Quick Start

1
Install dependencies 
pip install -r requirements.txt
2
Pull an Ollama model one-time 
ollama pull llama3.2
3
Train from your database one-time 
python train.py --db_host=localhost --db_user=root --db_pass=1234 --db_name=mydb
4
Start the API server 
python app.py
5
Open this page or embed the widget anywhere 
<script src="chat-widget.js"></script>
<script>
  ChatWidget.init({ apiUrl: "http://localhost:8000/ask" });
</script>

API Reference

MethodEndpointDescription
POST /ask Ask a question. Body: {"question": "...", "top_k": 5}
GET /health Check if the vector store is loaded and the server is ready
GET /chunks Browse raw knowledge chunks (debug). Params: limit, offset
GET /docs Interactive Swagger UI
